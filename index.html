Démo Light/Dark Mode fontion CSS light-dark()

À propos de cette démo

Cette page utilise la nouvelle fonction CSS light-dark() pour gérer les couleurs en fonction du thème. Le thème par défaut suit vos préférences système, mais vous pouvez le modifier avec le bouton ci-dessus.

Caractéristiques

  • Utilisation de light-dark() pour les couleurs
  • Transitions fluides entre les thèmes
  • Support des préférences système
  • Animations et effets interactifs
  • Bouton de basculement manuel
